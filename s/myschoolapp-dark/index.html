this is a dark theme for myschoolapp.com. when installed, it:

  • restyles all colors to the base16 color scheme, white on black (like this page)
  • normalizes fonts so that only one font is used across the entire site
  • removes awfulness such as animations, shadow effects, rounded corners, etc.

(download myschoolapp-dark.css above, install with Stylish for Firefox or Stylish for Chrome (untested))

the second file is a collection of several improvements to the behavior of the site:

  • reduces the number of clicks to change assignment status from 3 to 1, like so:
  • automatically sorts assignments by due date, the way most sane human beings would find to be actually useful
  • replaces distracting animations on topics pages with still images (sorry Dr. Harris)

(download myschoolapp-patches.user.js, install with Greasemonkey for Firefox or Tampermonkey for Chrome (untested))
